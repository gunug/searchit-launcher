# SearchIt Launcher - Google AI Studio TTS 대본

아래 대본은 Google AI Studio TTS에 바로 입력하기 위한 버전입니다. 예시처럼 감정 태그를 문장 앞에 붙였고, 화면 지시와 시간 표기는 제외했습니다.

---

## Korean - Version B

```text
[gentle] 앱 정리, 은근히 피곤하죠? [tired] 정리해도 금방 흐트러지고, 결국 또 찾고 또 정리하게 됩니다. [relief] 서치잇은 내 사용 습관을 배워, 앱 순서를 알아서 맞춰줘요. [clear] 필요한 앱은 고정하고, 안 쓰는 앱은 자연스럽게 아래로. [confident] 정리 스트레스를 줄이는 런처. SearchIt Launcher. [bonus] 참, 무료예요. 광고도 없고요.
```

## 추천 사용

- 한국어 TTS가 "SearchIt"을 어색하게 읽으면 `Korean - Version B, 조금 더 자연스러운 발음용`을 사용합니다.
- 영어 TTS가 "SearchIt"을 한 단어로 이상하게 읽으면 `English Version, pronunciation-safe`를 사용합니다.
- Bonus 문장이 20초 안에 들어가지 않으면 별도 컷, 설명란, 고정 댓글용 내레이션으로 분리합니다.
