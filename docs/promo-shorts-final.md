# SearchIt Launcher — 채택 대본 (5번 · 한/영 TTS 스크립트)

**컨셉**: 사용 습관 학습 — 쓸수록 알아서 정리되는 똑똑한 런처.
**길이**: 약 20초 세로 영상 (Shorts / Reels / TikTok)

> 큰따옴표 안 문장만 TTS가 읽습니다. 괄호 `( )` 안은 화면 연출 지시입니다.

>

> **(12~16초 / Body 3)**

## TTS 제작 메모

- **한국어**: 네이버 클로바 또는 구글 WaveNet 한국어 음성, 속도 0.95~1.0배. "서치잇"은 또렷하게 끊어 읽히는지 미리 확인.
- **English**: 자연스러운 미국식 음성 권장, 속도 1.0배. "SearchIt"은 한 단어로 붙여 읽도록 발음 확인(필요 시 "Search-it"으로 표기 테스트).
- 두 버전 모두 Hook 직후 BGM 비트가 들어오면 몰입도 상승. 첫 3초 안에 시연 화면으로 전환할 것.
