# 검색 순위 체계

검색 결과는 섹션 구분 없이 단일 목록으로 표시된다.  
아래 우선순위 순서대로 정렬하며, 앱 이름과 패키지명은 동일한 검색 대상으로 취급한다.

---

## 우선순위 (높음 → 낮음)

| 순위 | 분류 | 조건 |
|:----:|------|------|
| 1 | 완전일치 | 대소문자·띄어쓰기 포함 글자 전체 일치 |
| 2 | 완전일치 | 대소문자 무시, 띄어쓰기 포함 글자 전체 일치 |
| 3 | 완전일치 | 대소문자·띄어쓰기 무시 글자 전체 일치 |
| 4 | 부분일치 | 앞에서부터 연속 일치 (startsWith) |
| 5 | 부분일치 | 단어 경계에서 시작하는 연속 일치 |
| 6 | 부분일치 | 중간부터 연속 일치 (contains) |
| 7 | 초성일치 | 초성 전용 쿼리가 앱 이름 초성에 포함 |
| 8 | 이력일치 | 동일 검색어로 이전에 실행한 앱 (최근일수록 우선) |
| 9 | 자판오타일치 | 한/영 전환 없이 입력한 QWERTY 자판 배열 매칭 |
| 10 | 발음유사일치 | 영문 ↔ 한글 발음 유사, 로마자 변환 매칭, 영문 약어 |
| 11 | 추정일치 | 문자 중복도, 부분수열 매칭 |

---

## 동점 처리

같은 순위 안에서는 앱 이름 가나다 / 알파벳 오름차순으로 정렬한다.
