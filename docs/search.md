# 검색대상
* 앱의 이름
* 앱의 패키지명 중 도메인과 앱명칭 (com.google.sampleapp라면 google과 sampleapp)

# 우선순위
## 완전일치
* 대소문자포함 띄여쓰기포함 글자 전체 일치
* 대소문자무시 띄여쓰기포함 글자 전체 일치
* 대소문자무시 띄어쓰기무시 글자 전체 일치
## 부분일치
* 앞에서부터 연속으로 글자 일부 일치 (startsWith)
* 단어 경계에서 시작하는 연속 일치 (예: "play" → Google **Play**, Dis**play** 보다 우선)
* 중간부터 연속으로 글자 일부 일치 (contains)
## 초성일치
* 한글 초성만으로 구성된 검색어가 앱 이름의 초성 순서에 포함 (예: ㅋㅋ → 카카오톡)
## 이력일치
* 동일 검색어로 이전에 실행한 앱 우선 (최근 실행일수록 높은 순위)
## 자판오타일치
* 한/영 전환 없이 입력한 경우 자판 배열 기준으로 매칭 (예: rkrkdh → 카카오)
## 발음유사일치
* 영문으로 입력한 검색어가 한글 앱 이름의 발음과 유사한 경우 (예: instagram → 인스타그램)
* 한글로 입력한 검색어를 로마자로 변환했을 때 영문 앱 이름과 유사한 경우 (예: 티맵 → tmap)
* 영문 앱 이름의 첫 글자 조합 약어 (예: gp → Google Play)
## 추정일치
* 글자의 연속성은 무시 해당 알파벳이 많이 존재함
* 검색어의 글자가 순서를 유지하며 앱 이름에 분산되어 포함 — 부분수열 (예: ggl → google)
